# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.1.0] - 2026-04-04

### Added
- **Prompt Templates**: 40+ templates across 5 categories (portrait, landscape, anime, product, art)
- **Prompt Scoring System**: AI-powered quality assessment with clarity, completeness, creativity scores
- **Batch Generation**: Generate multiple prompts at once with comparison view
- **Version History**: Local storage for prompt history with search and favorites
- **Browser Extension**: Chrome/Edge/Firefox extension with right-click menu support
- **Keyboard Shortcuts**: Ctrl+Shift+P to quick-generate from selection
- **Vercel Deploy**: One-click deployment configuration
- **Export Options**: Export prompts in JSON, CSV, or TXT format

### Improved
- **README Documentation**: Complete documentation with examples
- **Technical Spec**: Detailed SPEC.md for developers
- **Project Structure**: Organized module structure

## [1.0.0] - 2026-03-21

### Added
- Initial release
- Text to image prompt generation
- Image to prompt (style extraction)
- Video script generation
- Multi-model support (Flux, DALL-E, Stable Diffusion, Runway, Veo)
- Custom API configuration
- Browser extension (basic)
- i18n support (Chinese/English)
- NextAuth authentication
